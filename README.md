# Flatbrainer Public Documents

This repository publishes public app documents and lightweight app information pages.

It is intended for stable URLs that can be used in app stores, support channels, release notes, and public project references.

## Published Sections

- App pages live under `apps/<app-name>/`.
- Privacy policies live next to the app they describe.
- Support pages provide public contact and troubleshooting information.
- General documents live under `documents/`.

## GitHub Pages

This repository is set up for GitHub Pages with Jekyll.

Recommended Pages settings:

- Source: deploy from a branch
- Branch: `main`
- Folder: `/`

Stable public URLs should use the page permalinks, for example:

- `/apps/7ime7racker/`
- `/apps/7ime7racker/privacy/`
- `/apps/7ime7racker/support/`
- `/apps/`
- `/documents/`
